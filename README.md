# socks5
socks5 proxy server builds with go/golang

### Features
* Both TCP and UDP supported
* "No Auth" authentication supported
* User/Password authentication supported

## Install
```
$ go get github.com/jqqjj/socks5
```

## Usage
* -p
  * port on listen (default 1080)
* -user (option)
  * username
* -pwd (option)
  * password
